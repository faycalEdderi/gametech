# GAMETECH

Projet Hitema <br>

Site d'actualité Gaming<br>

Developpement d'un interface administrateur/redacteur pour : <br>
  Publier un article<br>
  Modifier un article<br>
  Supprimer un article<br>
  Supprimer les commentaires indesirables<br>
  L'administrateur doit pouvoir se créer un compte et se connecter pour acceder a ces fonctions<br>
  <br><br>
  Developpement d'une vue :<br>
  Article contenant une photo du texte et un espace commentaire libre<br>
  Afficher les articles par categories<br>
  Afficher une barre de recherche fonctionnel<br>
  <br><br>
  Contien une BDD regroupant :<br>
  Les articles<br>
  Les categories<br>
  Les administrateurs (les seuls user du site)
