# GAMETECH

Projet Hitema 

Site d'actualité Gaming

Developpement d'un interface administrateur/redacteur pour : 
  Publier un article
  Modifier un article
  Supprimer un article
  Supprimer les commentaires indesirables
  L'administrateur doit pouvoir se créer un compte et se connecter pour acceder a ces fonctions
  
  Developpement d'une vue :
  Article contenant une photo du texte et un espace commentaire libre
  Afficher les articles par categories
  Afficher une barre de recherche fonctionnel
  
  Contien une BDD regroupant :
  Les articles
  Les categories
  Les administrateurs (les seuls user du site)
